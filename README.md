# Контроллер яркости монитора
## Позволяет быстро и удобно выставлять яркость на определённые уровни

1. Яркость - 25%
2. Яркость - 40%
3. Яркость - 55%
4. Яркость - 70%
5. Яркость - 85%

## Инициализация работы программы осуществляется путём запуска файла "bry.exe",
## находящегося в папке "/dist"
